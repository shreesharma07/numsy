🔢 Numsy Lightweight number processor
Open app API Health NPM GitHub

Process phone numbers with confidence.

Numsy is a lightweight TypeScript library + CLI that validates and sanitizes Indian phone numbers, and helps you parse CSV/Excel files with automatic phone-column detection.

Indian mobile validation 10 digits, starts with 6–9, smart sanitization.
CSV / Excel parsing Detects phone columns and supports bulk processing.
CLI + Web UI Run locally with npx @numsy/numsy-serve.

Quick start

1) Install Use your favorite package manager.
2) Validate Sanitize and validate Indian numbers in one call.
3) Process files Parse CSV/Excel and generate clean outputs + analytics.
TypeScript Docs 
import Numsy from 'numsy';

const numsy = new Numsy();

numsy.validate('+91-987-654-3210');
// { original, sanitized: '9876543210', isValid: true }
